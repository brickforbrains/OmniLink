# OmniLink: The Systems Archeologist

**A Logic-Based Puzzle Adventure built with Godot 4 \& C#**

## Project Overview

You play as a Junior Systems Archeologist at OmniCorp, tasked with "debugging" a legacy OS that powers the world. To progress, you must explore "Directory Dungeons" and repair broken circuits using core computer science principles.

## Tech Stack \& Architecture

* **Engine:** Godot 4.x (.NET Version)
* **Language:** C# (Modern .NET 8/10 standards)
* **Design Patterns:** Event-Driven Architecture (Signals), State Machines, and Strategy Pattern for Logic Gates.
* **TDD:** Unit testing via GDUnit4 to ensure logic-gate accuracy.
